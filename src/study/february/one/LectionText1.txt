В java мы все пишем камелькейсом, рассмотрим HelloWorld, System - класс, out - поле,
JDK включает в себя JRE, (java runtime environment)
Отступы в джава ровно два пробела, лол
Скомпилили .java файл с помощью javac, получили .class файл, запускаем его с помощью java. Компилятор внутри jvm классифицируется как JIT компилятор
.java -> .class -> .code
В си  .c (gcc)-> .0 (ln) -> .elf(.code)
В байткод можно скомпилировать из scala, groovy, kotlin еще, например.
Примитивы содержать фиксированное количество байт, но физическое их количество зависит исключтельно от реализации на
конкретном процессоре (если 32-битный инт запускаем на проце, который адресует по 64, то инт будет 64, хотя вести
будет себя вести как 32)
Примитивы: byte, boolean, char, short, int, long, float, double
Также есть какой-нибудь BigInteger, BigDecimal

стрингу, разрезать сплитом,
java mycode.class "1 2     -6" 3
должна выдавать 1+2-6+3
